## Run Instructions

```bash
npm i

npm run dev
```
## Description
A website has been developed to display a list of cryptocurrencies based on rank using Next.js, with Zustand for state management and IndexedDB to enhance UX.

## Library

- axios : For better management of endpoints in the project.
- dexie :‌ Better management of IndexedDB in the project.
- iconsax-react :‌ Show icons in project
- usehooks-ts : Using the useInterval hook due to an issue React has with setInterval.
- zustand :‌ State Management in project
